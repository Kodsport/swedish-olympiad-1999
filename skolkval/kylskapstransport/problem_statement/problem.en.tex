\problemname{Fridge Transport}

A factory that manufactures refrigerators needs to deliver a large shipment of $n$ fridges to a supermarket. The factory has two trucks at its disposal.

\begin{itemize}
  \item Truck $A$ costs $p_a$ kr/trip and can carry $k_a$ fridges at a time.
  \item Truck $B$ costs $p_b$ kr/trip and can carry $k_b$ fridges at a time.
\end{itemize}

Your task is to write a program that, given the five variables above, determines how many trips each truck should make in order to minimize the \emph{total transport cost}.

\section*{Input}
The input consists of the five integers $p_a$, $k_a$, $p_b$, $k_b$ ($500 \leq p_a, p_b \leq 2000$, $10 \leq k_a, k_b \leq 50$) and $n$ ($1 \leq n \leq 1000$) on a single line, separated by spaces.

\section*{Output}
Print three integers: the number of trips truck $A$ should make, the number of trips truck $B$ should make, and the total cost in kronor.
For all given test cases, it is guaranteed that the answer is unique.

\section*{Scoring}
Your solution will be tested on a set of test groups.
To earn points for a group, you must pass all test cases in that group.
